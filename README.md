# Journal

A secure, encrypted journal for the terminal, written in Python and powered by Textual.

Journal provides a clean terminal interface for writing, reading, and managing
private journal entries. All data is encrypted and protected by a master passwor
d, ensuring your thoughts remain accessible only to you.

## Features

* 🔒 Password-protected journal
* 🔐 Encrypted journal storage
* 📝 Create and edit entries
* 📖 Browse previous entries
* 🔍 Search journal entries
* 📅 Automatic timestamps
* 💻 Modern terminal user interface built with Textual

## Installation

```bash
git clone https://github.com/python-neo/journal.git
cd journal
pip install -e .
```

## Running

```bash
journal
```

## Requirements

* Python 3.12 or later
